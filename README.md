# Weather-Dashboard

## User Story

```md
AS AN everyday citizen
I WANT to know the weather in my city
SO THAT I can plan my outfit accordingly
As A traveller 
I WANT to be able to search the weather in other cities 
IN ORDER to effectively plan my trip
```

## Usage

```md
In order to use the Weather Dashboard, type any city into the search bar and
you will be given the weather forecast for today and the rest of the week.
Your recent searches are also saved to localStorage.
```
## Build
```md
This app runs in the browser and features dynamically updated HTML and CSS powered by jQuery. 
It uses OpenWeatherAPI to pull the weather data and stores user entered data to localStorage.

Link: http://127.0.0.1:5500/Weather-Dashboard/index.html
